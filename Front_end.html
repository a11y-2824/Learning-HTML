What is Front-end Development

Front end devlopment is using the UI interace to help the users interact with the webpage

The commonly used languages are HTML, CSS and Javascript

Front end developers build the front part of the website

They work with UI/UX designers and back end developers

Why is front end important

It provides a digital representation of the company

It acts as a bridge between customer and company

It is the realization of a dream
